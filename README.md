   1- 브라우저는 요청만<br>

3. Stateless<br>
   1- 요청1과 요청2는 의존성이 없어야함<br>
4. Cacheable<br>
   1- 서버에서 보내주는 정보들은 캐싱이 가능해야함<br>
   2- 캐싱을 위한 버전 같은 것도 관리 잘해야함<br>
